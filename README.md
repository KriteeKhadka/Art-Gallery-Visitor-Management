# Art Gallery Visitor Management System

A Python-based command-line application designed to manage visitor records and streamline operations within an art gallery environment.

## Overview

* Manages visitor information efficiently
* Provides a structured system for tracking entries
* Uses file handling for storing and updating records
* Operates through a simple menu-driven interface

## Features

* Visitor Management

  * Add new visitor details
  * View stored visitor records
  * Maintain organized data

* Record Handling

  * Store visitor data in files
  * Retrieve and display information when needed
  * Update records dynamically

* System Operations

  * Simple navigation through menu options
  * Input validation for better data accuracy

## Technologies Used

* Python
* File Handling (Read/Write)
* Modular Programming

## How to Run

* Clone the repository
  git clone https://github.com/your-username/art-gallery-visitor-management.git

* Navigate to the project directory
  cd art-gallery-visitor-management

* Run the application
  python main.py
